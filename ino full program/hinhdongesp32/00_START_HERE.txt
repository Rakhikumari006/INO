📊 TECHNICAL SPECIFICATIONS

Platform
  • Microcontroller: ESP8266 (NodeMCU, Wemos D1 Mini, etc.)
  • Display: SSD1306 OLED 128×64 pixels
  • Interface: I2C @ 400kHz
  • Address: 0x3C

Performance
  • Frame Rate: 20-50 FPS (adjustable)
  • Button Response: < 100ms
  • Display Update: ~20ms per frame
  • Memory Flash: 1.0MB (250KB code + 772KB frames)
  • Memory RAM: 3KB objects, 48KB available
  • Power Draw: 100mA normal, 20mA sleep

Components
  • AnimationEngine: 6 emotional states with progress tracking
  • DisplayManager: OLED control with power optimization
  • InputHandler: Debounced button with event detection
  • StateMachine: 5-state device with auto-transitions
  • ConfigSystem: Centralized settings (no code edits needed!)

🎯 CUSTOMIZATION EXAMPLES

Change Animation Speed
  Edit config.h:
    #define IDLE_SPEED 20        // Faster (default: 42)
    #define SLEEPY_SPEED 80      // Slower

Change Sleep Timeout
  Edit config.h:
    #define SLEEP_TIMEOUT 60000  // 60 seconds (default: 30)

Adjust Brightness
  Edit config.h:
    #define BRIGHTNESS_SLEEP 25  // Dimmer sleep mode

Add Custom Emotion
  Edit animations.h, add in setEmotion():
    case NERVOUS:
      currentStartFrame = 421;
      currentEndFrame = 470;
      animationSpeed = 35;
      isLooping = true;
      break;

Change Frame Ranges
  Edit config.h:
    #define FRAME_HAPPY_START 101
    #define FRAME_HAPPY_END 200

All done in config.h - no complex code editing needed!
